先声明一下，我之前几乎只用过 IntelliJ IDEA。

它真的挺强的，重构、导航、调试、插件，我可太喜欢了  
问题说起来挺好笑的，以前我总觉得它太贵，长期用破解脚本，但是我总感觉对不起这个强大的 IDE

后来我就转去 Eclipse 了

我当时选择 Eclipse 的原因是：

1. FOSS
2. 在我这个垃圾电脑上，内存比 IDEA 小一些
3. 快捷键我很喜欢
4. 界面没有太多花哨的东西

但是它的很多东西我很不喜欢：

1. 搞半天原来要自己拼
   Java 包默认很基础，如果项目有点复杂的话，还要装插件和调配置什么的
2. 代码补全有点垃圾  
   它代码补全很不符合我的习惯，不如 IDEA
3. 如果我插件装多了，这就是个 IDE 版 Optifine(
   维护成本太大了。。。

我那段时间感受是：Eclipse 能凑合写，也免费，但是如果我想让它更适合我，估计要调配置八百年了。。。

然后在我已经准备说“免费 IDE 就没个好的吗”的时候，IDEA 推出了合并版，我又把 IDEA 下回来了

装回来那天，我先卸了 Eclipse，启动 IDEA，内存就炸了
下一步我又把快捷键改成 Eclipse 风格的了

因为 Eclipse 的快捷键，真的很好用

过了几个月，我想明白一件事：  
不同的 IDE 可能并不适合全部人，还需要自己拼
我现在用的是 IDEA，挺好用的，我现在主要开发 MInecraft mod了，它插件里的 MInecraft mod 开发插件也挺好用的
